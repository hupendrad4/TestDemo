# Changelog

All notable changes to TestDemo will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release of TestDemo
- User authentication and authorization with JWT
- Role-based access control (Admin, Test Manager, Tester, Developer, Viewer)
- Test Project management
- Test Suite (hierarchical) management
- Test Case management with steps
- Test Plan and Build management
- Test Cycle management (Smoke, Sanity, Regression, etc.)
- Test Execution tracking
- Test Step execution with results
- Requirements management
- Requirements-to-Test Case linking
- Defect management
- Defect-to-Execution linking
- Custom fields support
- Keywords/Tags for test cases
- File attachments for test cases, executions, and defects
- Comment system for defects
- Dashboard with metrics
- Basic reporting module
- RESTful API with Swagger documentation
- Comprehensive database schema with Prisma ORM
- Rate limiting and security features
- Logging system with Winston
- React frontend with Material-UI
- Redux state management
- Responsive design
- Docker support with docker-compose
- API documentation
- Installation guide
